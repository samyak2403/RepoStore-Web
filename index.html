Everything you need, nothing you don't

RepoStore turns GitHub releases into a clean, Play Store style experience on Android.

Smart Discovery

Home sections for Trending, Recently Updated, and Featured projects. Only repos with valid APK assets are shown — zero source-only noise.

Latest-Release Installs

Fetches the latest release for each repo. Contains a single Install action with download progress delegating to the system installer.

Update Tracking

Tracks installations and detects when updates are available. Shows an Open button for installed apps to keep you in the loop.

From discovery to install in seconds

A seamless pipeline built entirely on top of the public GitHub API.

1

Search & Discovery

Uses GitHub's API to find projects meeting strict conditions based on topics and languages.

2

3

Install

Stay Updated

Get your app listed

No manual submission needed. RepoStore uses the public GitHub Search API — if your project meets these conditions, it appears automatically.

  • Public repository on GitHub
  • At least one published, non-draft release
  • Installable assets (.apk)
  • Relevant topics and description for discoverability
